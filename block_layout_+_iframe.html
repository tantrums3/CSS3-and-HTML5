The code tags that create blocks and iframes can be found at the W3Schools Website under the lessons for HTML. Tags that are useful in creating block layouts include:

* the < div > tag coupled with a [ class ] or
* the combination of specific tags such as:
- < main >
- < header >
- < nav >
- < aside >
- < section >
- < article >
- < quote > & < cite >
- < footer >
- < address >
- < details > & < summary >
- and many more.
